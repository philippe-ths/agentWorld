# Agent World

A simple Phaser 3 isometric game. A player moves around a 64x64 tile map with 3 NPCs that wander randomly.

## Tech Stack

- **Phaser 3** — game engine
- **TypeScript** — language
- **Vite** — bundler / dev server

## Run

```bash
npm install
npm run dev      # http://localhost:8080
npm run build    # production build → dist/
```

## Controls

Arrow keys or WASD to move the player one tile at a time.
